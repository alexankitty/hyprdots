# Alexankitty's hyprland dot files
Vaxry gave me entirely too much power over what the compositor may do, and so I have taken to abusing it because I have problems.

## Features
Loads everything in the config folder automagically.

Has a high level of granularity so it's easy to find what piece of config is handling what.

Features my app-mgmt library I used for handling uwsm, delays, runonce, and replace.

Features a command builder to make it easier to expand the functionality of the command manager.

Features a dispatch wrapper for compatibility with hl.bind and hl.dispatch such that they'll receive the requisite function as a closure that can be fully ran.

Features my never-ending insanity, pls send help.
